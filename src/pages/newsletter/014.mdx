## üî• Headlines

<WebGPULogo /> [**WebGPU ships in Chrome 113**](https://developer.chrome.com/blog/webgpu-release/)
(currently in the Beta channel). This release marks a new era for GPU-accelerated graphics on the
web. <ThreeLogo /> Three.js, <BabylonLogo /> Babylon.js, and <PlayCanvasLogo /> PlayCanvas all have
some initial support for WebGPU, but expect a bumpy road if you want to support both WebGL and
WebGPU, considering that WebGPU uses WGSL instead of GLSL for shaders. The transition will be rough,
but the future is bright!

<PhaserLogo /> [**Phaser 3.60.0 is
out!**](https://github.com/photonstorm/phaser/releases/tag/v3.60.0) This massive update has been in
the works for over a year. It also marks the [10th
anniversary](https://github.com/photonstorm/phaser/graphs/contributors) of the first Phaser release
on GitHub on April 12th, 2013. The most recent beta versions include the
[Timeline](https://www.patreon.com/posts/phaser-v3-60-22-80835382) class and a
[Matter.js](https://github.com/liabru/matter-js) v0.19 update. Check out the complete
[**Changelog**](https://github.com/photonstorm/phaser/releases/tag/v3.60.0) of this major update!

[Anslo](https://x.com/anslogen) shared his thoughts on JavaScript game development and explained some of the performance optimizations and techniques used in [Slow Roads](https://slowroads.io/) in an excellent [**tech article on Google's web.dev**](https://web.dev/slow-roads/) website.

We are one step closer to The Matrix scenario with this [**paper of a simulation**](https://x.com/emollick/status/1645499660402925576) involving 25 AI-driven characters hanging out in a virtual town. [Watch the replay](https://reverie.herokuapp.com/arXiv_Demo/) while wondering who is currently watching yours.
