## üî• Headlines

<WebGPULogo /> **WebGPU** support landed in [**Safari 26.0**](https://webkit.org/blog/17333/webkit-features-in-safari-26-0/#webgpu)! And any site can be [installed as a web app](https://webkit.org/blog/17333/webkit-features-in-safari-26-0/#every-site-can-be-a-web-app-on-ios-and-ipados) on iOS and iPadOS.

üò∫ [**Navcat**](https://github.com/isaac-mason/navcat/) by [Isaac Mason](https://x.com/isaac_mason_) (who made [recast-navigation-js](https://github.com/isaac-mason/recast-navigation-js)) is a new full-featured **navmesh** and **pathfinding** library for your game AI agent needs. Check out the many [**examples**](https://navcat.dev/). It's already used in production in Mana Blade! Isaac is also cooking up an ecosystem of cat-themed libraries for games, like [packcat](https://github.com/isaac-mason/packcat) and [mathcat](https://github.com/isaac-mason/mathcat).

üéæ [**Bounce**](https://perplex.gg/bounce/) ([repo](https://codeberg.org/perplexdotgg/bounce)) by [Perplex](https://x.com/perplexdotgg) (our dear HamzaKubba on [Discord](https://webgamedev.com/discord)) is a new **physics library**. Weighing only 60kb gzipped, it's a batteries-included library with spatial queries, fast raycasting, sweep and intersection tests. It is renderer-agnostic, 100% deterministic, and does not require WebAssembly. Check out the [demos](https://perplex.gg/bounce/#demos) and [API](https://perplex.gg/bounce/#api).

<R3FLogo /> [**Triplex**](https://triplex.dev/), the React Three Fiber **editor** by [Douges](https://x.com/dougesdev), went fully [open source and moved to the pmndrs](https://triplex.dev/blog/triplex-moves-to-pmndrs) collective!

„Ä∞Ô∏è [**Dot Noise**](https://mini.gmshaders.com/p/dot-noise) ([demo](https://www.shadertoy.com/view/wfsyRX)) by [Xor](https://x.com/XorDev) is a new cheap alternative to 3D simplex noise. I implemented it for TSL in [TSLFX](https://tslfx.v1v2.io/misc/dot-noise).
