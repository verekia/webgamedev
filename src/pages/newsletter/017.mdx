## üî• Headlines

A few **game jams** are coming up soon:

The first [**React Jam**](https://reactjam.com/) (organized by [Rune](https://developers.rune.ai/)) is starting **tomorrow**. It's a 10-day event, with $2500 of cash prizes, and I will be judging submissions! You can make <CssLogo /> DOM-based games with divs and CSS, 2D canvas games with libraries like <PixiLogo /> [Pixi React](https://pixijs.io/pixi-react/), or 3D games with <R3FLogo /> [React Three Fiber](https://docs.pmnd.rs/react-three-fiber), for example.

[**Multiplayer Madness**](https://itch.io/jam/multiplayer-madness-by-hathora) (organized by [Hathora](https://hathora.dev/)) is a 10-day game jam starting on **August 11th** with $2500+ in prizes and focused on online multiplayer games. Use any engine, web or native. [Steven Yau](https://www.webgamedev.com/interviews/steven-yau) is part of the jury!

[**js13kGames**](https://js13kgames.com/), the competition with a file size limit of 13kb running yearly since 2012, will start on **August 13th**. You might want to look into [Kontra.js](https://straker.github.io/kontra/) for that one.
