[**Safari 16.4 is out!**](https://webkit.org/blog/13966/webkit-features-in-safari-16-4/) The most relevant features for us are:

- ‚Ä¢ Third-party web browsers Add to Home Screen for üåê [**PWAs**](https://www.webgamedev.com/publishing/pwa) (called _Home Screen web apps_ by Apple).
- ‚Ä¢ [**Web Push**](https://developer.mozilla.org/en-US/docs/Web/API/Push_API) on iOS and iPadOS ‚Äì Installed PWAs can send notifications.
- ‚Ä¢ [**Badging API**](https://developer.mozilla.org/en-US/docs/Web/API/Badging_API) ‚Äì Set a badge on the installed PWA icon to act as a notification.
- ‚Ä¢ [**Offscreen Canvas**](https://developer.mozilla.org/en-US/docs/Web/API/OffscreenCanvas) ‚Äì 2D-only for now.
- ‚Ä¢ [**Fullscreen API**](https://developer.mozilla.org/en-US/docs/Web/API/Fullscreen_API) on iPadOS ‚Äì Reportedly unsuitable for games because swiping down exits fullscreen.
- ‚Ä¢ [**Screen Orientation API**](https://developer.mozilla.org/en-US/docs/Web/API/ScreenOrientation) ‚Äì Preliminary support, can't trigger a rotation yet.
- ‚Ä¢ [**Screen Wake Lock API**](https://developer.mozilla.org/en-US/docs/Web/API/Screen_Wake_Lock_API) ‚Äì Keep your game open without the screen turning off.
- ‚Ä¢ [**Compression Streams API**](https://developer.mozilla.org/en-US/docs/Web/API/Compression_Streams_API) ‚Äì Compress and decompress gzip directly from JavaScript.
- ‚Ä¢ [**Growable** `SharedArrayBuffer`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/SharedArrayBuffer#growing_sharedarraybuffers) and [**resizable** `ArrayBuffer`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/ArrayBuffer).
- ‚Ä¢ [**Media Queries Level 4 Syntax Improvements**](https://developer.mozilla.org/en-US/docs/Web/CSS/Media_Queries/Using_media_queries#syntax_improvements_in_level_4) ‚Äì For example: `@media (400px <= width < 900px)`
- ‚Ä¢ [**Import Maps**](https://web.dev/import-maps-in-all-modern-browsers/) ‚Äì Useful to map ES module URLs to names if you don't use a bundler.

üìê A mathematical breakthrough happened: the first ever aperiodic monotile called [**the hat**](https://cs.uwaterloo.ca/~csk/hat/) has been discovered. It is a single shape that tiles a plane, but never periodically ‚Äì meaning it cannot be translated to align with itself. This pattern might have interesting applications in game development, who knows! Here is a [Veritasium video](https://www.youtube.com/watch?v=48sCx-wBs34) about infinite patterns covering the Penrose tiling (which is made of 2 shapes combined).
