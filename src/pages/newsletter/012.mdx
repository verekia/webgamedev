## üî• Headlines

<ThreeLogo /> [0beqz](https://twitter.com/0beqz) released his highly-anticipated postprocessing
library for Three.js, [**Realism Effects**](https://twitter.com/0beqz/status/1632109961768476672)
([repo](https://github.com/0beqz/realism-effects), [demo](https://realism-effects.vercel.app/)). The
effects include SSGI (Screen-Space Global Illumination), motion blur, and TRAA (Temporal
Reprojection
Anti-Aliasing). <R3FLogo /> [react-postprocessing](https://github.com/pmndrs/react-postprocessing)
integration should be coming soon.

[**Self-service publishing**](https://www.unrealengine.com/en-US/blog/self-service-publishing-now-available-for-the-epic-games-store) is now available for the [Epic Games Store](https://store.epicgames.com/) ([more details](https://store.epicgames.com/en-US/news/epic-games-store-launches-self-publishing-tools-for-game-developers-and-publishers)). There is a $100 submission fee per game, and you can get an [IARC](https://en.wikipedia.org/wiki/International_Age_Rating_Coalition) rating usable in other stores. Here is a [**video walkthrough by Epic**](https://www.youtube.com/live/dV8L5pto39k?feature=share&t=544).

<R3FLogo /> [Cody Bennett](https://twitter.com/Cody_J_Bennett) published
[**preact-reconcilier**](https://twitter.com/Cody_J_Bennett/status/1632674026408558592). It
implements [react-reconciler](https://www.npmjs.com/package/react-reconciler), which allows custom
renderers to be shared between [Preact](https://preactjs.com/) and React. The reconciler makes it
possible to use the React Three Fiber ecosystem including [Drei](https://github.com/pmndrs/drei)
with Preact and [Preact Signals](https://twitter.com/Cody_J_Bennett/status/1633063536158023680).
