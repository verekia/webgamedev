## 🔥 Headlines

<BabylonLogo /> [**Babylon.js 9.0**](https://blogs.windows.com/windowsdeveloper/2026/03/26/announcing-babylon-js-9-0/)
by [Babylon.js](https://x.com/babylonjs) – Clustered Lighting (hundreds of dynamic lights on WebGPU + WebGL 2),
Volumetric Lighting via compute shaders, Node Particle Editor, and Frame Graph v1 with 40%+ GPU memory savings.
Also adds Animation Retargeting across skeletons and expanded Gaussian Splat support.

<BabylonLogo /> [**Babylon Lite**](https://forum.babylonjs.com/t/introducing-babylon-lite/63648)
([repo](https://github.com/BabylonJS/Babylon-Lite)) by [Deltakosh](https://x.com/deltakosh) – WebGPU-exclusive
ground-up redesign of Babylon.js: 19x less gzipped JS, 3.6x faster with pixel-perfect identical output.
Functional no-classes API with WGSL minification via Vite. Demo scenes: DOOM, LibreQuake, and Tetris.

<ThreeLogo /> [**Three.js Conf**](https://threejs.paris) is the first-ever Three.js conference,

- <ThreeLogo /> [**ez-tree**](https://github.com/dgreenheck/ez-tree) – Procedural Three.js tree generator, now with rounded normals on leaves by [Dan Greenheck](https://x.com/dangreenheck)
- <GltfLogo /> [**Basis Universal Transcoder**](https://github.com/hwei/Basis-Universal-Transcoder-Project) ([demo](https://hwei.github.io/Basis-Universal-Transcoder-Project/)) – WASM KTX2 transcoder with TS bindings, eval-free by [hwei](https://github.com/hwei)
- <BlenderLogo /> [**jsblender**](https://github.com/verekia/jsblender) – Typesafe .blend file parser for Blender 5+: meshes, materials, lights, cameras, armatures, and custom properties by [me](https://x.com/verekia)
- <ThreeLogo /> [**AVAO**](https://avao.verekia.com/) ([repo](https://github.com/verekia/avao)) – Adaptive Vertex AO: AO baked into a re-meshed structure, vertices cluster at occlusion creases, one byte per vertex, no lightmap by [me](https://x.com/verekia)
- <ThreeLogo /> [**Anime.js 4.5 Three.js Adapter**](https://animejs.com/documentation/adapters/threejs-adapter/) – CSS transform-like API for 3D object animations, instanced mesh support, and stagger 3D by [Julian Garnier](https://x.com/JulianGarnier)
- <R3FLogo /> [**Ecctrl 2.0**](https://github.com/pmndrs/ecctrl) – Physics-driven character controller toolkit for R3F + Rapier by [Andrew Chen](https://x.com/AndrewChenE)
