## Theatre.js

<Project {...theatre} /> is an animation library and toolkit to create cinematic scenes and animating
objects of your scene. It contains a visual editor that can transform objects of your scene, which can
be quite useful for debugging and positioning objects. It supports [vanilla Three.js](https://www.theatrejs.com/docs/latest/getting-started/with-three-js)
and [React Three Fiber](https://www.theatrejs.com/docs/latest/getting-started/with-react-three-fiber).
