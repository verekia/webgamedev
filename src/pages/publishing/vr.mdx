# 🥽 VR

Web games can be played in VR through the Meta Quest browser, which supports WebXR. For wider distribution, you can submit your game to the Meta Quest Store (which now supports [PWAs](https://developers.meta.com/horizon/documentation/web/pwa-overview/)) or distribute it via SideQuest for sideloading.

## Resources

- [Meta Quest Store](https://developer.oculus.com/quest-store/)
